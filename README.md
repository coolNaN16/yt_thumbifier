# unstablifier

a browser extension that makes every thumbnail **UNSTABLE**

adds an unstable smp character to every youtube thumbnail. there are 25 different variations

how to use:
- unarchive the zip file to a folder
- go to chrome://extensions/
- press "load unpacked" on the top left
- select the folder with unstablifier
- enjoy...

WILL BE UPDATED WHEN THERE ARE NEW UNSTABLE SMP VIDEOS
